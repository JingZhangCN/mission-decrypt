# Mission: Decrypt! A Hands-On Math Adventure 🕵️‍♂️🔑

Welcome to the official repository for **Mission: Decrypt!**, an interactive, browser-based cryptography workshop delivered as part of the **HKBU Summer Programmes for High School Students 2026**.

This program is custom-designed for international high school students to explore the intersection of mathematics, computer science, and practical security without requiring any local software setup or coding background.

## 🗺️ Workshop Syllabus & Terminals

The curriculum is structured into four mandatory interactive missions and a gateway hub:

### 🏠 [Gateway Hub (index.html)](index.html)
The starting landing page where students can access all individual simulator terminals.

### 🟢 [Section 1: Telegraph Simulator (1_morse_code.html)](1_morse_code.html)
* **Mathematical Concept:** Encoding vs. Encryption, Signal mapping.
* **Hands-on Task:** Students tap on an interactive telegraph key (or use the Spacebar) with real-time sound feedback to translate their names and countries into binary Morse signals.

### 🔵 [Section 2: Caesar Decryption Wheel (2_caesar_cipher_wheel.html)](2_caesar_cipher_wheel.html)
* **Mathematical Concept:** Caesar Cipher shift functions, Brute-force limitations, and Modular Arithmetic ("Clock Math" $x \pmod{26}$).
* **Hands-on Task:** Students spin a virtual inner/outer wheel to decrypt Roman military orders (`ATTACK`) before formalizing the mathematics of key-shifts.

### 🟠 [Section 3: RSA Asymmetric Encryption (3_rsa.html)](3_rsa.html)
* **Mathematical Concept:** Asymmetric Key Pairs (Public vs. Private keys), One-way prime modulo arithmetic ($c = m^e \pmod n$).
* **Hands-on Task:** Alice encrypts a fruit emoji using Bob's public key; Bob decrypts the secret cargo using his private key in a simulated public transmission line.

### 🟣 [Section 4: Digital Signature (4_digital_signature.html)](4_digital_signature.html)
* **Mathematical Concept:** Document integrity, One-way Hash functions, Avalanche Effect, and Verification algorithms ($s = h^d \pmod n$).
* **Hands-on Task:** Students create custom digital hash spaces (grids) to sign documents and test how modifying a single character instantly invalidates verification.
